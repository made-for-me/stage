# Sample static data

This directory is the static data layout the viewer expects.

## Layout

```text
sample-data/
  README.md
  screen-map.v1.json
  session-ref.v1.json
  screens/
    home.json
    settings.json
```

## How to use it

- Copy the JSON files generated by `stage` and `stage-runner` into this shape.
- Keep screenshot references relative so the same bundle can be hosted locally or in CI.
- Treat these files as the canonical static example for the viewer and prompts.
