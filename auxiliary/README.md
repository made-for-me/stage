# Auxiliary Repo Blueprints

These directories are split-ready blueprints for the repos described in the Stage OSS plan.

- `stage-runner/` defines the self-hosted remote-session API and worker responsibilities.
- `stage-devtools/` defines the open tooling layer that consumes `stage` manifests and `stage-runner` sessions.

They stay in-tree until their release cadence justifies separate repositories.
